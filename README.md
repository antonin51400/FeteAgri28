# FeteAgri28

Site statique de pilotage pour la Fête de l'Agriculture de la Marne 2028, organisée par les Jeunes Agriculteurs du canton de Suippes.

## Modules

- Dashboard avec scénarios prudent / réaliste / ambitieux
- Gestion admin locale avec ajout, édition et suppression
- Bénévoles, exposants, tâches, budget, concert, rider Bazalt et checklist
- Vues par rôle : admin, président, commissaire général, trésorier, responsable commission, bénévole, exposant, sponsor, visiteur
- Drag & drop sur listes et cartes
- Sauvegarde locale dans le navigateur via localStorage

## Déploiement

Le site est statique : `index.html`, `styles.css`, `app.js` et `assets/hero-agri-festival.svg`.

Compatible avec :

- GitHub Pages
- Vercel
- Netlify

Pour GitHub Pages : Settings > Pages > Deploy from branch > `main` > `/root`.

## Note

Cette version GitHub utilise un fond SVG léger et déployable. La version locale créée dans Codex conserve aussi le grand visuel PNG généré.
